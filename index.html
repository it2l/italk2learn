Welcome to iTalk2Learn project.

iTalk2Learn is a learning platform for children aged between 8 and 12 years old. It is an open source platform which integrates structured practice and exploratory, conceptually-oriented learning. It facilitates interaction in different modalities including speech, as well as multiple representations which can be manipulated and reasoned with when learning fractions.

Installation

Compiling the iTalk2Learn system itself can be done using

mvn install

This generates a "war" file which can be deployed by Tomcat 7

Some modules are not public, please contact to request more info.

Instructions

It is required to have a database and LDAP running. The project is based on MariaDB 5.5 and ApacheDS. Files to populate can found here
