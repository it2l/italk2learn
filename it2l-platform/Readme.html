Installation

Required hardware (minimum)

  • INTEL compatible CPU, 2 cores, 1.6GHz, SSE2
  • 3 GB DDR3 RAM
  • Disk space: 60 GB

Required software

  • Supported: Windows 8 (64 bit), Windows 7 (64 bit), Windows Vista (64 bit), Windows XP (64 bit), Windows 2008 R2 Server
  • Java Runtime Environment (32 bits). For the LMT feature, Java Runtime Environment (JRE) 1.7 or superior is required. Please verify that it is installed by executing the following command in a Command Window: 
    java -version
    and that the environment variable that points to the JRE installation is correctly set: 
    echo %JAVA_HOME%
    or 
    echo %JRE_HOME%
    Remember use 32 bit version to compile
    To download JRE for Windows, please download it from: Java Downloads for All Operating Systems
    To install Java please refer to the official documentation in: How do I install Java?
  • Maven 3.0. To install Maven please refer to the official documentation in: How do I install Maven?
  • Apache Tomcat 7(32 bits). To install Apache Tomcat please refer to the official documentation in: How do I install Apache Tomcat?
    Choose 32 bits version and unzip in folder where you want to deploy the project

Installation steps

  • Configuring Tomcat 7.
    Default Tomcat port is 8080
    To start Tomcat go to /bin/startup.bat
    Add following users to your tomcat in /conf/tomcat-users.html 
    		  <role rolename="admin-gui"/>
		  <role rolename="manager-gui"/>
		  <role rolename="manager"/>
		  <role rolename="admin"/>
		  <role rolename="manager-script"/>
		  <user password="script" roles="manager-script" username="script"/>
		  <user username="admin" password="admin" roles="admin,manager,admin-gui,manager-gui"/>
  • Configuring Maven 3.
    To set Maven go to official documentation in: How do I configure Maven?
  • Downloading the project.
    Go to Italk2learn official github on: https://github.com/italk2learnlbl/italk2learn Download zip file or add project to your git repository.
  • Access to Birkbeck VPN.
    In order to have access to our database and LDAP you will need to connect to Birkbeck VPN
    Send an email to: jlfg@dcs.bbk.ac.uk

Compilation steps

  • Go to project root folder and execute (remember starts Tomcat first): 
     mvn clean compile package tomcat:deploy
    To undeploy project in Tomcat execute:
     mvn tomcat:undeploy

Running the platform

  • To access to Italk2learn platform click http://localhost:8080/italk2learn/
