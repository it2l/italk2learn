Welcome to Italk2learn

This sample demonstrates how to use Spring MVC and Spring Web Flow together with Thymeleaf and Tiles.

Italk2learn platform in this sample include:

  • A sequence of whizz exercises with a shell container developed by whizz
  • A sequence of fraction tutor exercises

You can go to the next exercise just only clicking "Next Exercise" Button

The exercise sequence is retrieved of the recommender system and it depends of the user that has been logged on Italk2learn Platform
